# -Rapaduras-Artesanais-Prot-tipo-Flutter
Este protótipo demonstra um catálogo digital de produtos artesanais:  Visualização de produtos com imagem, descrição, peso e preço.  Adicionar ao Carrinho com feedback em tempo real (Snackbar).  Carrinho integrado no AppBar com contador dinâmico.  Design clean, rústico (marrom, creme) e totalmente responsivo.
